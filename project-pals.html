Optimizing PALS Placement in Elk Creek

A GIS-based assessment and recommendation strategy for enhancing Bull Trout habitat using Post-Assisted Log Structures.

Project Overview

Elk Creek, Alberta, provides critical habitat for threatened Bull Trout but faces degradation from land use and artificial structures. Post-Assisted Log Structures (PALS) are used in restoration efforts by Freshwater Canada. This project aimed to assess the effectiveness of existing PALS and identify optimal locations for additional structures by leveraging GIS analysis to maximize habitat benefits. The core objective was to develop a data-driven strategy for PALS placement focusing on enhancing stream complexity, sinuosity, and woody debris recruitment within a target reach downstream of a constricting bridge.

My Role & Process

As the sole researcher for this class project, my responsibilities included:

  • Creating and preparing a LiDAR derived DEM for hydrological analysis.
  • Performing hydrological analysis using ArcGIS Pro (Flow Direction, Flow Accumulation, Slope, Curvature).
  • Calculating Stream Power Index (SPI) to assess erosion potential.
  • Synthesizing analysis results to create an Erosion Susceptibility map identifying high-risk banks.
  • Developing a PALS placement strategy based on analysis, literature, and project goals (targeting high erosion zones near wood sources).
  • Running basic 3D flow simulations to visualize potential impacts of recommended PALS.
  • Creating map layouts and an interactive ArcGIS StoryMap to present the methodology, results, and recommendations.

Tools & Technologies

  • GIS Software: ArcGIS Pro (Hydrology Toolset, Surface Toolset)
  • Web GIS: ArcGIS Online, ArcGIS StoryMaps
  • Data Sources: Sawback Technologies, Freshwater Canada, 2024, [Elk Creek LiDAR Point Cloud Data], [Elk Creek Orthographic Data].

Results & Key Findings

The analysis indicated that while existing PALS are contributing positively, specific opportunities exist for enhancement. High-risk erosion zones coinciding with significant stream power and favorable bank shapes were identified. The recommended PALS placements target these zones to strategically leverage natural stream energy for increased habitat complexity, sinuosity, and wood recruitment. Flow simulations provided visual support, suggesting the recommended PALS would increase flow interaction with targeted banks. The full results and detailed recommendations are presented in the linked StoryMap.
