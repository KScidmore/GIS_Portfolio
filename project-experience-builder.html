Elk Creek: Hydrology and Morphology Explorer

An interactive web application for exploring hydrological analysis results in the Elk Creek study area.

App Purpose & Use Cases

The primary purpose of this application was to provide an interactive platform for exploring the spatial datasets generated during the Elk Creek PALS placement analysis. It allows users to visualize and compare hydrological and morphological layers like Stream Power Index, slope, curvature, flow patterns, elevation, and orthoimagery directly within a web browser. The app includes an introductory card that also links to the full analysis StoryMap for further context.

Potential users could use this app to:

  • Get a brief overview and link directly to the comprehensive StoryMap analysis via the introductory Card widget.
  • Visually correlate different datasets (e.g., high SPI vs. steep slopes).
  • Understand the real-world context using orthoimagery and basemaps.
  • Explore the study area dynamically by panning, zooming, and toggling layers.
  • Utilize measurement tools for basic distance/area estimations.
  • Serve as a visual aid for presentations or discussions about the Elk Creek analysis.

Development Process & Widgets

This application was built using ArcGIS Experience Builder. The core process involved:

  1. Creating the map.
  2. Choosing a suitable template for the task.
  3. Adding and configuring widgets for user interaction:
    • Card: Provides introductory text and a prominent button linking directly to the full project StoryMap.
    • Layer List: Allows users to toggle data layer visibility.
    • Legend: Enables interpretation of map symbology for visible layers.
    • Basemap Gallery: Offers options for changing the background map.
    • Coordinates: Displays the geographic coordinates of the cursor location.
    • Measurement: Provides tools for calculating distances and areas.
  4. Arranging these widgets within the application layout.

Data Layers Included

The application allows exploration of the following datasets hosted in ArcGIS Online:

  • Stream Power Index (SPI) Raster
  • Slope Raster
  • Curvature Raster
  • Flow Accumulation Raster
  • Flow Direction Raster
  • Digital Elevation Model (DEM)
  • Orthoimagery
  • Standard ArcGIS Online Basemaps
