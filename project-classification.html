Supervised Land Cover Classification

Performing supervised image classification and accuracy assessment for land cover mapping using ArcGIS Pro.

Final land cover classification map using a Support Vector Machine classifier.

Project Overview

This lab project simulated a real-world scenario requiring the production and evaluation of a land cover classification map. The primary objective was to perform a supervised, pixel-based classification on provided multispectral imagery and conduct a thorough accuracy assessment to evaluate the reliability of the results. This involved defining training data, running the classification algorithm, refining the results, and analyzing classification accuracy using a confusion matrix.

My Role & Process

As the sole analyst for this lab, my responsibilities covered the entire workflow:

  • Generating a classification schema and creating representative training area polygons for various land cover types (Water, Gravel, Bare Ground, Shrub, Grassland, Forest).
  • Executing a supervised pixel-based classification using the Support Vector Machine (SVM) algorithm within the ArcGIS Pro Image Classification Wizard.
  • Performing initial classification runs and refining the results through manual reclassification or merging of similar classes.
  • Creating accuracy assessment points and assigning ground truth values through visual inspection.
  • Conducting a quantitative accuracy assessment using the validation points and the classified map.
  • Generating and interpreting a confusion matrix, including calculating overall, producer's, and user's accuracies.
  • Documenting the process, identifying challenges (including initial attempts and issues with object-based classification), and suggesting potential improvements.

Tools & Technologies

  • GIS Software: ArcGIS Pro
  • Classifier: Support Vector Machine (SVM)
  • Key Tools: Image Classification Wizard, Create Accuracy Assessment Points, Compute Confusion Matrix
  • Data Sources: Provided orthographic imagery (`Lab.tif`)
  • Other Software: Microsoft Excel (for confusion matrix analysis/visualization)

Results & Key Findings

The primary objectives were met, resulting in a final pixel-based land cover map and a detailed accuracy assessment. The accuracy assessment revealed varied performance across different land cover classes. Gravel showed perfect user's accuracy, indicating high reliability when classified, although some gravel pixels were missed (lower producer's accuracy). Water was also classified relatively well. However, significant confusion occurred between Forest, Shrub, and Grassland classes, with Forest showing particularly low producer's and user's accuracies. This confusion was likely due to spectral similarities and variability within these vegetation classes. The accuracy assessment itself was limited by the number and visual interpretation of validation points, and the inherent limitations of pixel-based methods were acknowledged. Recommendations for improvement included using more comprehensive training data and potentially object-based classification methods (if software instability could be resolved).
